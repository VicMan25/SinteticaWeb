# ⚽ Sistema de Reservas de Canchas

Proyecto final para la aplicación de conocimientos en desarrollo de software web utilizando **Java JEE 8**.  
El sistema permite la gestión integral de **clientes, canchas y reservas** mediante un CRUD funcional.

---

## 🛠️ Tecnologías y Requisitos

### 🔹 Desarrollo
- **Lenguaje:** Java  
- **Tecnología Web:** JSP y Servlets  
- **IDE:** NetBeans  
- **Servidor:** Apache Tomcat 9.0  
- **Base de Datos:** MySQL (Workbench)  
- **Frontend:** Bootstrap 5, HTML, CSS y JavaScript  
- **Gestión de Dependencias:** Maven  

### 🔹 Ejecución
- **Sistema Operativo:** Windows  
- **Navegador:** Google Chrome  
- **Conexión:** MySQL Local  

---

## 📋 Requerimientos Funcionales

El sistema cuenta con los siguientes módulos principales:

### 👤 Gestión de Clientes
- Registro de usuarios  
- Edición de información  
- Eliminación de clientes  

### 🏟️ Gestión de Canchas
- Administración de canchas  
- Tipos de cancha  
- Asignación de precios  

### 📅 Gestión de Reservas
- Programación de fechas y horas  
- Control de disponibilidad  
- Estados de reserva  

---

## 🏗️ Diseño del Software

### 📊 Modelo Entidad-Relación
El sistema implementa una relación **Muchos a Muchos (N:M)** entre **Clientes** y **Canchas**, utilizando la tabla intermedia de **Reservas**.
